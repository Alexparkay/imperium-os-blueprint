# Memory Index

Navigation hub for the knowledge base. The registry (`.claude/reference/registry.md`) catalogs the system; this file maps the memory layer.

- `kanban.md` + `calendar.md` - tasks, commitments, dates (the live board)
- `people.md` + `people/` - relationships, dossiers, frameworks
- `owner_profile.md` - the owner
- `patterns.md` - lessons and behavioral patterns
- `finances.md` - financial records (figures only with provenance)
- `business/` - strategy notes and product docs
- `content/` - ideas backlog, creator blueprints
- `raw/` - drop zone for unprocessed sources
- `braindumps/` - long-form voice-dump archives (router audit logs live in `.context/braindump/`)
- `system_changelog.md` - how the OS itself evolves
