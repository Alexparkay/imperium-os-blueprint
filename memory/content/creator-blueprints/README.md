# Creator Blueprints

One folder per creator you study deeply: `<creator-name>/` containing a master blueprint (their frameworks, hooks, voice patterns, what makes their content work) plus the raw transcripts it was built from. Built by the creator-deep-dive workflow: pull their videos via `automations/youtube/`, extract transcripts, analyse, write the blueprint. Start with 1-2 creators whose audience overlaps {{ICP}}; depth beats breadth.
