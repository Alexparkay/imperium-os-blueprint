# Content Memory

The knowledge side of content (the production side lives in `content-pipeline/`). This folder holds the idea backlog (`ideas-backlog.md`, create on first use), publishing history, content strategy docs, and `creator-blueprints/` for studies of creators worth learning from. Rule of thumb: if it's ABOUT content (ideas, strategy, analysis) it lives here; if it IS content (drafts, finals) it lives in `content-pipeline/`.
