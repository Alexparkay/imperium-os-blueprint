# MANDATORY: Persist Knowledge Immediately

When the owner mentions ANY new information during conversation:

- People, contacts, relationships → `memory/people.md` (+ dossier in `memory/people/` if substantial)
- Goals, strategy changes, decisions → `memory/business/` (or `memory/kanban.md` for commitments)
- Content ideas, hooks, titles → `memory/content/`
- Revenue, costs, financial → `memory/finances.md` (rule 31 provenance applies)
- Behavioral patterns, lessons → `memory/patterns.md`
- Dates, deadlines, events → `memory/calendar.md`

Write it IMMEDIATELY. Do not wait until end of session. Do not wait to be asked.

When updating one memory file, check whether related files also need updating (a new person mentioned → update `memory/people.md` AND check whether a dossier exists).
