# templates/

Fill-in skeletons, one per content format. Start every draft by copying the matching template into `../drafts/`; the embedded comments carry the structural rules so you don't have to remember them. Each template ends with a style checklist: run it before the piece moves anywhere.

- `blog-brief.md`: brief format for handing a blog post to a writer
- `content-repurposing.md`: turn one long-form piece into multi-platform assets
- `email-nurture.md`: 5-email nurture sequence skeleton
- `linkedin-post.md`: short-line, hook-first LinkedIn format
- `newsletter.md`: hook / insight / application / CTA newsletter format
- `youtube-script.md`: hook-scripted, bullet-riffed video format

Add new templates as new formats enter the rotation; keep the checklist pattern.
